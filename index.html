vix

AI coding agent powered by free LLMs

Windows (PowerShell)

irm https://vix.codes/install.ps1 | iex

macOS / Linux

curl -fsSL https://vix.codes/install.sh | bash

Get started

  1. Get a free API key at Google AI Studio
  2. Set your key: export GOOGLE_API_KEY=your-key
  3. Run: vix --model gemma

Supported models

gemma — Gemma 4 31B
gemini — Gemini Flash
llama — Llama 3.3 70B
claude — Claude Sonnet
gpt-4o — GPT-4o
qwen — Qwen (local)
